# Code of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of behavior that help:

- being specific and technical in feedback
- assuming good intent while reviewing code
- welcoming different levels of experience
- focusing criticism on the work, not the person

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language or conduct
- trolling, intimidation, or deliberate disruption
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions
that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers
through GitHub rather than escalating in a public thread when possible.
